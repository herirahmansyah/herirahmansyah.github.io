Project 1 – Website Portofolio

Mengubah Data Mentah Menjadi Insight Bisnis dengan Power BI Baru saja menyelesaikan proyek analisis data terbaru saya menggunakan Microsoft Power BI. Dalam proyek ini, saya menganalisis 4 tahun data penjualan retail (Superstore Dataset) untuk menjawab pertanyaan bisnis seputar tren penjualan dan performa wilayah.

Proses & Insight Utama: # Data Cleaning (ETL): Menggunakan Power Query untuk menstandarisasi tipe data dan memastikan akurasi. # DAX Measures: Membuat kalkulasi kustom untuk melacak Total Sales ($2.3M) secara dinamis. # Analisis Tren: Menggunakan fitur drill-down untuk melihat pola musiman (seasonality) dari tahunan hingga bulanan. # Top Performer Analysis: Mengidentifikasi 10 Negara Bagian dengan penjualan tertinggi untuk strategi alokasi sumber daya. Dashboard ini dirancang agar interaktif, memungkinkan pengguna memfilter data berdasarkan Wilayah (Region) untuk melihat detail yang lebih spesifik.
