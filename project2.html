Project 2 – Automasi Data dengan n8n

Automasi Proses Data Menggunakan n8n

Project ini berfokus pada automasi alur kerja data menggunakan n8n untuk mengintegrasikan berbagai sumber data dan meningkatkan efisiensi operasional.

Proses & Insight Utama

  • Workflow Automation: Membuat workflow otomatis untuk pengambilan, pemrosesan, dan distribusi data.
  • Integrasi API: Menghubungkan berbagai layanan seperti Google Sheets, Google Drive, Gmail, dan OpenAI.
  • Error Handling: Menangani error otomatis untuk menjaga kestabilan sistem.

Proyek ini dibuat untuk membantu HR atau tim SDM rekrutmen untuk menilai CV pelamar secara cepat dan objektif dengan Bantuan AI. Adapun fungsinya Membuat sistem berbasis AI yang menganalisis CV pelamar kerja dan memberikan penilaian otomatis.
